<h3>Description:</h3>

The Pong Game is a classic and addictive arcade game built with Python and Pygame. challenge a friend or play against an AI opponent. 
<br>

<h3>Players:</h3>

Up to 2 Players
<ul>
  <li>Player 1: W, S</li>
  <li>Player 2: UP, DOWN</li>
</ul>

<h3>Difficulty:</h3>

3 Levels of difficulty:
<ul>
  <li>Easy</li>
  <li>Medium</li>
  <li>Hard</li>
</ul>

<h3>Lives:</h3>

Each player gets 5 lives.
